# Лабораторная работа №2

## Проект
**Разработка системы оценки риска диабета 2 типа**

## Назначение Lab 2
Lab 2 переводит проект из стадии анализа (Lab 1) в стадию проектирования решения и подготовки прототипа.
В рамках работы подготовлены архитектурные материалы, инженерные заметки, план работ и минимальные каркасы backend/frontend для демонстрации базового сценария оценки риска.

Система рассматривается как учебный веб-инструмент **предварительной** оценки риска и не является медицинской диагностической системой.

## Быстрый локальный запуск
1. Backend:
   - Перейти в папку `backend`
   - Установить зависимости: `pip install -r requirements.txt`
   - Запустить API: `uvicorn app.main:app --reload --port 8000`
2. Frontend:
   - Перейти в папку `frontend`
   - Запустить статический сервер: `python -m http.server 5500`
   - Открыть `http://localhost:5500`

По умолчанию frontend обращается к API по адресу `http://localhost:8000`.
