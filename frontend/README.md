# Frontend (Lab 2)

Минимальный интерфейс анкеты для учебной системы оценки риска диабета 2 типа.

## Локальный запуск
1. Перейти в папку `frontend`
2. Запустить:
   - `python -m http.server 5500`
3. Открыть в браузере:
   - `http://localhost:5500`

По умолчанию frontend отправляет запросы на `http://localhost:8000`.
