# RACI-матрица (студенческая команда)

Обозначения: **R** — Responsible (исполняет), **A** — Accountable (несет ответственность), **C** — Consulted (консультирует), **I** — Informed (информируется).

| Задача | SA/PO | Fullstack developer | AI engineer | Tester |
|---|---|---|---|---|
| Сбор требований | A | R | C | I |
| Подготовка документации | A | R | C | I |
| Проектирование системы | A | R | C | C |
| Разработка интерфейса | C | A/R | I | C |
| Разработка backend | C | A/R | C | I |
| Подготовка AI-логики | C | C | A/R | I |
| Тестирование | I | C | C | A/R |
| Подготовка отчета | A | R | C | C |
