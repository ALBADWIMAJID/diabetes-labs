# PRD — Лабораторная работа 1

## Название проекта

### Веб-система предварительной оценки риска диабета 2 типа

## Описание проблемы
Многие люди поздно обращают внимание на факторы риска диабета 2 типа и не проходят своевременную проверку.  
Нужен простой учебный инструмент, который помогает оценить риск на основе анкеты и мотивирует пользователя обратиться за профессиональной медицинской консультацией при повышенном риске.

## Цель системы

Создать веб-систему для предварительной оценки риска диабета 2 типа по набору базовых параметров пользователя с выводом понятного результата и кратких рекомендаций.

## Целевая аудитория

- Взрослые пользователи, которые хотят пройти первичную самооценку риска.
- Студенты и преподаватели, использующие систему как учебный проект.
- Администратор проекта, который поддерживает параметры анкеты и контролирует работу системы.

## Основные пользовательские потребности

- Быстро пройти анкету в одном окне без сложной регистрации.
- Получить понятный уровень риска (низкий/средний/высокий).
- Понять, какие факторы повлияли на итог.
- Получить базовые рекомендации по дальнейшим действиям.
- При необходимости сохранить результат в истории пользователя.

## Основные функции системы

- Ввод данных пользователя через веб-форму (возраст, ИМТ, семейный анамнез, физическая активность и др.).
- Расчет итогового балла риска по простой балльной модели.
- Определение категории риска: низкий, средний, высокий.
- Отображение результата и факторов, которые повлияли на оценку.
- Вывод кратких рекомендаций по профилактике и обращению к специалисту.
- Базовая административная настройка вопросов анкеты и порогов категорий риска.

## Ограничения MVP

- Система **не ставит диагноз** и не заменяет консультацию врача.
- Используется упрощенная учебная модель расчета риска.
- Нет интеграции с медицинскими информационными системами.
- Нет автоматической передачи данных в клиники.
- Минимальная аналитика, без сложной персонализации.

## Критерии успеха

- Не менее 80% тестовых пользователей проходят анкету за 5 минут.
- Для подготовленных тест-кейсов система корректно определяет категорию риска.
- Результат и рекомендации формулируются понятно для пользователя без специальной подготовки.
- Документация Lab 1 принята преподавателем без критичных замечаний.

## Нефункциональные требования

- Веб-интерфейс работает в актуальных версиях Chrome, Edge и Firefox.
- Время ответа при расчете результата: до 2 секунд при обычной нагрузке.
- Интерфейс должен быть простым и читаемым (понятные поля, подписи, кнопки).
- Данные пользователя обрабатываются с базовыми мерами защиты (ограниченный доступ, без публикации персональных данных).
