# PRD — Лабораторная работа 1

## Название проекта
**Веб-система предварительной оценки риска диабета 2 типа**

## Описание проблемы
Часть пользователей не оценивает личные факторы риска диабета 2 типа на раннем этапе и откладывает профилактические действия.
Учебному проекту нужен простой инструмент первичной самооценки, который помогает понять уровень риска и при необходимости обратиться к врачу.

## Цель системы
Создать понятную веб-систему для предварительной оценки риска диабета 2 типа на основе короткой анкеты и выдачи кратких рекомендаций.

## Целевая аудитория
- Взрослые пользователи, которые хотят пройти первичную самооценку риска.
- Студенты и преподаватели, использующие систему как учебный проект.
- Команда проекта, поддерживающая корректность анкеты и расчетных правил.

## Основные пользовательские потребности
- Быстро заполнить анкету без сложных действий.
- Получить понятный уровень риска (низкий/средний/высокий).
- Понять, какие факторы повлияли на итог.
- Получить базовые рекомендации по дальнейшим действиям.

## Основные функции системы
- Ввод данных через веб-форму (возраст, рост/вес, физическая активность и др.).
- Расчет итогового балла по прозрачной rule-based модели.
- Определение категории риска: низкий, средний, высокий.
- Отображение результата и краткого пояснения по факторам риска.
- Вывод базовых рекомендаций профилактического характера.

## Ограничения MVP
- Система **не ставит диагноз** и не заменяет консультацию врача.
- Используется упрощенная учебная модель расчета риска.
- Нет авторизации, личного кабинета и хранения истории.
- Нет интеграции с медицинскими информационными системами.

## Критерии успеха
- Не менее 80% тестовых пользователей проходят анкету за 5 минут.
- Для подготовленных тест-кейсов система корректно определяет категорию риска.
- Результат и рекомендации понятны пользователю без специальной подготовки.
- Документация Lab 1 принята преподавателем без критичных замечаний.

## Нефункциональные требования
- Веб-интерфейс работает в актуальных версиях Chrome, Edge и Firefox.
- Время ответа при расчете результата: до 2 секунд при обычной нагрузке.
- Интерфейс простой и читаемый (понятные поля, подписи, кнопки).
- Данные запроса обрабатываются только в рамках учебного сценария без публикации персональных данных.
